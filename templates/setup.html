⚙️ Google 試算表設定

3 分鐘搞定！免 Cloud Console、免服務帳戶 🎉

📝 步驟說明

1

建立 Google 試算表

前往 Google 試算表，建立一個新的試算表，名稱取 護照掃描紀錄。

📌 把試算表的網址複製下來存好，等下會用到
2

打開 Apps Script 編輯器

在試算表中，點選上方選單的 「延伸功能」→「Apps Script」。

它會在新分頁打開一個程式碼編輯器，把裡面預設的程式碼全部刪掉。

3

貼上這段程式碼

把下面的程式碼全部複製，貼到 Apps Script 編輯器裡，完全覆蓋原本的內容：

/** * 小柴子護照掃描器 - Google 試算表 Webhook */ function doPost(e) { try { const data = JSON.parse(e.postData.contents); const sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet(); if (sheet.getLastRow() === 0) { const headers = [ '掃描時間', '姓氏', '名字', '護照號碼', '國籍', '出生日期', '性別', '有效期限', '個人ID', '簽發日期', '出生地', '簽發機關', 'MRZ第一行', 'MRZ第二行', '原始OCR文字' ]; sheet.getRange(1, 1, 1, headers.length).setValues([headers]); sheet.getRange(1, 1, 1, headers.length).setFontWeight('bold'); } const row = [ data.掃描時間 || new Date().toLocaleString('zh-TW', {timeZone:'Asia/Taipei'}), data.姓氏 || '', data.名字 || '', data.護照號碼 || '', data.國籍 || '', data.出生日期 || '', data.性別 || '', data.有效期限 || '', data.個人ID || '', data.簽發日期 || '', data.出生地 || '', data.簽發機關 || '', data.MRZ第一行 || '', data.MRZ第二行 || '', (data.原始OCR文字 || '').substring(0, 500) ]; sheet.appendRow(row); return ContentService .createTextOutput(JSON.stringify({ success: true })) .setMimeType(ContentService.MimeType.JSON); } catch (err) { return ContentService .createTextOutput(JSON.stringify({ success: false, error: err.toString() })) .setMimeType(ContentService.MimeType.JSON); } } function doGet() { const sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet(); const data = sheet.getDataRange().getValues(); let html = '<h2>📋 護照掃描紀錄</h2><p>共 ' + (data.length-1) + ' 筆資料</p>'; return HtmlService.createHtmlOutput(html); }
4

部署為網頁應用程式

在 Apps Script 編輯器中：

① 按左上角的 「部署」 → 「新增部署作業」

② 類型選 「網頁應用程式」

③ 「執行身分」 選 「我」（你的 Google 帳號）

④ 「誰可以存取」 選 「任何人」

⑤ 按 「部署」

⚠️ 第一次部署時會跳出授權畫面，按「檢閱權限」→ 選你的 Google 帳號 → 「進階」→ 「前往...」→ 「允許」
5

複製網址 & 設定到 App

部署完成後，會出現一個網址（長得像 https://script.google.com/macros/s/.../exec）。

把這個網址複製下來。

✅ 然後到 Render 的 Dashboard → 你的服務 → Environment → 新增變數：

GOOGLE_SHEET_WEBHOOK_URL = 你剛剛複製的那串網址

儲存後，Render 會自動重新啟動 App，右上角就會出現 ✅ Google 試算表已連線！
