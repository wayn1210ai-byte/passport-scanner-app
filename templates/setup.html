⚙️ Google 試算表設定

← 回到掃描器

想讓掃描的資料自動寫入 Google 試算表嗎？跟著以下步驟，5 分鐘搞定！

1 建立 Google Cloud 專案

前往 Google Cloud Console，登入你的 Google 帳號，點擊頂端選單建立一個新專案（名稱隨便取，例如「passport-scanner」）。

2 啟用 Google Sheets API

在左側選單 → 「API 和服務」 → 「程式庫」，搜尋 Google Sheets API，點進去按「啟用」。

3 建立服務帳戶

左側選單 → 「API 和服務」 → 「憑證」 → 上方點「建立憑證」 → 「服務帳戶」。

名稱填 passport-scanner-sa，角色可以先不用選，按「完成」。

4 產生金鑰

在服務帳戶列表找到剛建立的帳戶，點擊進入。切到「金鑰」分頁 → 「新增金鑰」 → 「JSON」。

瀏覽器會自動下載一個 .json 檔案，這就是你的私鑰！

📁 把這個 JSON 檔案存好，等一下要上傳到伺服器

5 建立 Google 試算表

前往 Google 試算表，建立一個新的試算表。名稱建議取 護照掃描紀錄。

點擊右上角「共用」，加入剛才的服務帳戶 Email（在 JSON 金鑰檔裡面的 client_email 欄位），權限設為「編輯者」。

📌 把試算表的網址記下來，裡面的 ID 長這樣：1BxiMVs0XRA5nFMdKvBdBZjgmUUqptlbs74OgVE2upms

6 設定到 App 中

將以下兩個環境變數設定到你的伺服器：

GOOGLE_SHEET_CREDENTIALS_JSON = (把 JSON 金鑰檔的內容整段貼進來) GOOGLE_SHEET_ID = 你的試算表ID（上面網址裡那一串）

設定完成後重新啟動 App，右上角就會出現 ✅ Google 試算表已連線！

不想設定 Google 試算表？沒關係！

App 還是可以正常掃描護照，只是資料不會自動儲存喔～
